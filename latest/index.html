Install

Consumer applications install the published package from npm and pair it with the token contract.

npm install @fuggetlenfe/components @fuggetlenfe/tokens @fuggetlenfe/brand-styles

Contract

  • Core logic lives in web components.
  • Brand identity lives in the external CSS token layer.
  • React and Angular wrappers are published separately.
  • Documentation is versioned with the released package.
